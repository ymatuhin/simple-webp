Simple-webp

View on GitHub Download .zip Download .tar.gz

About

This micro lib replace images with attr webp to webp image if webp supported. And if image is loaded â€” do nothing, else change src to .webp.

Usage

Via Bower

bower install simple-webp --save

or download the zip.

Put in header

  <script async src="simple-webp.min.js"></script>

Wrap your image in noscript with data-webp attribute

  <noscript data-webp><img src="example.jpg" alt=""></noscript>
