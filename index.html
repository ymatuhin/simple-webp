Simple-webp

About

This micro lib replace images with attr webp to webp image if webp supported. And if image is loaded â€” do nothing, else change src to .webp.

Examples

html

Put in header

  <script async src="webpjs.min.js"></script>

Add webp attr to image which have webp version

  <img src="example.jpg" alt="" webp>
